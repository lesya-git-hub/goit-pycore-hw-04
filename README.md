# Консольний бот-помічник (Version 1.0)

Цей проєкт є першою версією персонального консольного асистента, написаного на Python. Бот призначений для керування книгою контактів: додавання, зміни, видалення та перегляду збережених номерів.

## 🛠 Основні можливості
* **Додавання контактів**: Зберігає ім'я та номер телефону.
* **Зміна номерів**: Оновлює телефон для існуючого імені.
* **Видалення**: Видаляє запис з книги за ім'ям.
* **Пошук**: Виводить список усіх збережених контактів.
* **Відмовостійкість**: Бот має вбудований парсер, який обробляє помилки введення та не валиться при порожніх командах.
* **Збереження даних**: Усі контакти автоматично зберігаються у файл `contact_list.txt`.
